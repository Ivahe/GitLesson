# This text about config from cmd
### Заходим в папку с работой и вверху(пкти файла) пишем cmd
### git config --global user.email (пишем свой емаил)
### git config --global user.name (пишем свое имя)
### На каждый файл пишется git add ***.html
### git status
### git -m commit "комментарий"
### git push (загружаем на гит)



## Dir - показывает файлы в текущей папке
## CD.. - выходит из папки
## CD имя_папки - заходит тв папку с этим именем
## mkdir имя_папки - создает папку с этиим именем
## rmdir - удаляет папку
## copy con имя_файла/папки - удаляет
## help - команды список
## cls - очищает экран
## color 02 - меняет цвет
## tree - показывает дерево папок
## type - показывает содержимое файла
## del имя_файла - удаляет файл
